# 13-Layer Full-Stack Architecture LMS

An interactive learning management system built from the original static architecture handbook.

## What changed

- Split the interface into a documentation pane and a coding sandbox.
- Added Arabic coding challenges for all 13 architecture layers.
- Added a sandboxed JavaScript runner with per-test reporting.
- Added local persistence for code drafts, completed challenges, and theme state.
- Added a gamified progress bar and completion checkpoints.

## Challenge types

- Algorithm
- Debug
- Architecture decision

## How to use

1. Open the site in a browser.
2. Select a layer on the left.
3. Read the Arabic problem statement and hint.
4. Edit the starter code.
5. Run tests or submit the decision challenge.

## Notes

- Challenge progress is stored in `localStorage`.
- The repo still contains the original documentation tree under `docs/` for reference.
- The app is a static front-end and can be deployed directly to GitHub Pages.
